Spacing

An 8px base unit creates a consistent rhythm across all layouts, components, and screens.

Foundation

Spacing scale

All spacing values are multiples of 4px, anchored to an 8px base. Use named tokens, not raw pixel values, in component code.

Spacing and accessibility

  • check_circle All interactive elements must have a minimum touch target of 44×44px. Standard buttons are 48px tall; large CTAs are 58px. Never reduce button height below 44px. 2.5.5 AAA / best practice
  • check_circle Adjacent tap targets must have at least 8px of space between them to prevent accidental mis-taps, particularly for users with motor impairments. 2.5.8 AA (WCAG 2.2)
  • check_circle Radio button and checkbox rows must have a full-width tap area, not just the control itself. Extend the touch area to at least 44px height using padding on the label. 2.5.5
  • check_circle Icon-only buttons (notification bell, avatar) must have a minimum 44×44px tap area even when the visible icon is 24px. Use padding to expand the target. 2.5.5
  • check_circle Paragraph spacing must be at least 2× the font size. Line height must be at least 1.5× the font size. These values meet WCAG 1.4.12 text spacing requirements. 1.4.12 AA
