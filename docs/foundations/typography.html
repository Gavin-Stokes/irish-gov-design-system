Usage guidelines

Section label Personal details Enter your details as they appear on your MyGovID.
check_circle
Do use sentence case for all labels, headings, and body copy.
PERSONAL DETAILS Enter Your Details As They Appear On Your MyGovID.
cancel
Don't use ALL CAPS for headings or Title Case for body text.
info
Typography choices directly affect readability for people with dyslexia, low vision, and cognitive differences. Lato was chosen for its humanist letterforms and generous x-height.

Accessibility requirements

  • check_circle Body text is 16px minimum. Do not use body copy smaller than 16px — caption (12px) and supporting (14px) styles are for supplementary, non-critical content only. 1.4.4 AA
  • check_circle Line height for body text is 1.5× the font size. Do not compress line height below this for body copy. 1.4.12 AA
  • check_circle Letter spacing on body text is 0.005em. This is within WCAG 1.4.12 spacing requirements. 1.4.12 AA
  • check_circle Never use px units to set font sizes in production CSS — use rem so users who resize browser text see fonts scale correctly. The token values in colors_and_type.css use rem. 1.4.4 AA
  • check_circle Heading levels must reflect document hierarchy: one <h1> per page, then <h2>, <h3> in order. Never skip levels for visual sizing — use CSS classes instead. 1.3.1 A
  • check_circle Do not use ALL CAPS or text-transform: uppercase for body content — it is harder to read and may confuse screen readers in some locales. 3.1.5 AAA
  • check_circle Avoid justified text (text-align: justify) — it creates uneven word spacing that is difficult for readers with dyslexia. 1.4.8 AAA

Heading hierarchy example

HTML 
<!-- One h1 per page -->
<h1>Social housing application</h1>

<!-- Section headings -->
<h2>Before you begin</h2>

  <!-- Sub-sections -->
  <h3>What you'll need</h3>
  <h3>About this application</h3>

<h2>Personal details</h2>

  <!-- Don't skip: h2 → h3, never h2 → h4 -->
  <h3>Contact information</h3>

CSS tokens

CSS 
--font-primary:  'Lato', sans-serif;

/* Weights */
--fw-light:    300;
--fw-regular:  400;
--fw-bold:     700;
--fw-black:    900;

/* Scale (rem) */
--text-hero:     2rem;      /* 32px */
--text-h1:       1.75rem;   /* 28px */
--text-h2:       1.5rem;    /* 24px */
--text-h3:       1.25rem;   /* 20px */
--text-h4:       1.125rem;  /* 18px */
--text-body:     1rem;      /* 16px */
--text-small:    0.875rem;  /* 14px */
--text-caption:  0.75rem;   /* 12px */
--text-overline: 0.625rem;  /* 10px */

/* Line heights */
--lh-heading: 1.25;
--lh-body:    1.5;
--lh-tight:   1.1;

/* Letter spacing */
--ls-body:     0.005em;
--ls-overline: 0.2em;
