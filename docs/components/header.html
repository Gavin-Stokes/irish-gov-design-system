Header

The app header identifies the service as a government product, anchors navigation, and presents key user context.

Component WCAG 2.1 AA

Variants

Standard header

Used on all screens except the home screen. Background: #008660 (Rich Green). Total height: ~56px visible + 24px status bar = ~80px.

Hero header

Home screen content

Header anatomy

  1. 1 Irish Government Harp mark — white SVG on green background
  2. 2 Page title (standard) or greeting (hero) — white text
  3. 3 Notification icon — 24px, white, 44×44px tap target
  4. 4 User avatar / profile — circular, 32px, 44×44px tap target
  5. 5 DPENDR overline label (hero only) — 10px, wide tracking

When to use

  • The hero header appears on the home/dashboard screen only. It communicates the user's identity and the service name prominently.
  • The standard header appears on all other screens — form pages, confirmation screens, profile pages.
  • The Harp mark must appear in every header. It signals to citizens that they are using an official government service.
  • check_circle Wrap the header in <header role="banner">. This provides the ARIA landmark used by screen reader users to skip or navigate to the header. 1.3.6 AAA / best practice
  • check_circle The Harp mark must be wrapped in a link to the home screen: <a href="/"> with aria-label="Irish Government — home". 2.4.4 AA
  • check_circle A skip link (<a href="#main-content">Skip to main content</a>) must be the first focusable element in the document — before the header. 2.4.1 A
  • check_circle The notification icon button must have aria-label="Notifications". If there is a badge count, include it in the label: aria-label="Notifications, 3 unread". 1.1.1 A
  • check_circle The profile/avatar button must have aria-label="Profile menu" or equivalent. 1.1.1 A
  • check_circle White text on #008660 header achieves 4.6:1 contrast — passing AA. White on #004D44 hero achieves 10.7:1. 1.4.3 AA
  • check_circle The hero overlay texture is decorative. Apply it as a CSS background-image with mix-blend-mode: overlay, not as an <img> element. 1.1.1 A

Standard header

HTML
<!-- Skip link: must come first in DOM -->
<a href="#main-content" class="skip-link">Skip to main content</a>

<header role="banner">
  <a href="/" aria-label="Irish Government — home">
    <!-- Harp SVG -->
    <span>Irish Government</span>
  </a>

  <nav aria-label="Utility">
    <button type="button" aria-label="Notifications, 3 unread">
      <span class="material-icons" aria-hidden="true">notifications</span>
      <span class="badge" aria-hidden="true">3</span>
    </button>
    <button type="button" aria-label="Profile menu">
      <span class="material-icons" aria-hidden="true">person</span>
    </button>
  </nav>
</header>

<main id="main-content">
  <h1>Page title</h1>
  <!-- content -->
</main>

Hero header

CSS
.hero-header {
  background: var(--color-bg-hero);  /* #004D44 */
  padding: 20px var(--page-margin-x) 24px;
  position: relative;
  overflow: hidden;
}

/* Decorative texture overlay */
.hero-header::before {
  content: '';
  position: absolute;
  inset: 0;
  background: url('../../assets/hero-overlay.png') center / cover;
  opacity: 0.8;
  mix-blend-mode: overlay;
  pointer-events: none;
}

.hero-header__greeting {
  font-size: var(--text-h1);
  font-weight: var(--fw-bold);
  color: var(--color-text-on-dark);
  position: relative;  /* above pseudo-element */
}

.hero-header__overline {
  font-size: var(--text-overline);
  letter-spacing: var(--ls-overline);
  text-transform: uppercase;
  color: var(--color-text-on-dark-muted);
}
