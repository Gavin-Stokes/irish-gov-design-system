Do's & Don'ts
check_circle DO use sentence case
"Start application", "Apply now", "What happens next?" — never ALL CAPS or Title Case for body copy.
cancel DON'T use colour alone for state
Always pair colour with an icon or text label. e.g. error fields need aria-describedby + visible error message, not just a red border.
check_circle DO provide visible focus styles
Focus ring: outline: 3px solid #FFBF47 (GOVUK-style gold focus). Never outline: none.
cancel DON'T use placeholder as label
Always use a visible <label> element. Placeholder text disappears on focus and is inaccessible to screen readers.
check_circle DO use sufficient touch targets
Minimum 44×44px for all interactive elements on mobile. Standard buttons are 48–58px height.
cancel DON'T rely on icons alone
Icon buttons must have aria-label or visible text label. e.g. the notification bell needs aria-label="Notifications".
info Language: Irish + English
Use lang="en" on <html>. Irish-language content (e.g. "An Garda Síochána") should use lang="ga" on the relevant span.
info WCAG 2.1 AA Target
All public-facing Irish Government digital services must meet WCAG 2.1 Level AA. Heading hierarchy, skip links, and keyboard navigation are required.
